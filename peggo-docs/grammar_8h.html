typedef struct grammar_st grammar_t

Represents a full PEG grammar with production rules and starting symbol.

This structure owns its starting expression and all its rules once constructed.

Look up the production rule for a non-terminal symbol in a grammar.

Performs a search of the rules associated with grammar. If no match is found, NULL is returned. If a rule is returned, grammar retains ownership of it.

Parameters
grammarThe grammar to use for the lookup
symbolThe symbol to search for
