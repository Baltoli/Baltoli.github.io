◆ parse()

parse_result_t* parse ( char *  source,
grammar_t *  grammar 
)

Parse a string using a PEG grammar.

This function is not thread safe, as grammar is stored in a global variable for implementation purposes. If multiple threads need to call the parser, they should do so in mutual exclusion.

Does not take ownership of source or grammar. They may both be used freely after a call to parse.

If a parse is successful, the object returned will be a SUCCESS variant with a parse_t member in its union. Similarly, if the parse fails then it will be an ERROR variant.

Parameters
sourceA source text to be parsed
grammarThe grammar with which to parse the source
