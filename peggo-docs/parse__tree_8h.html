◆ parse_t

typedef struct parse_st parse_t

Represents the result of a parse as a tree structure.

This structure owns all of the child nodes it contains.

Add a new child node to a parse tree.

This allocates memory for another child node in the tree, and performs a shallow copy of child into this memory. The node child should not be used after it has been passed to this method.
